# Lecture slides for Kernel Machines

## Prerequisites

Install [mise](https://mise.jdx.dev/getting-started.html), then use it to install Node.js and pnpm:
```bash
mise use node@25
mise use pnpm@10
```

## Installation

```bash
pnpm install
```

## Presenting

```bash
pnpm run dev
```
Visit http://localhost:3030

## Exporting PDF

```bash
pnpm run export
```

## Editing

Edit [slides.md](./slides.md) to modify the main slides.
Contents of each section are in the `pages` directory.

Learn more at [sli.dev](https://sli.dev/).
